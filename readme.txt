Make a fully functional accordion to display various contents professionally. 

== Description ==

An accordion is used to show and hide HTML content. This block plugin helps you to make a fully functional accordion to display various contents professionally. 


**[Demos](https://bblockswp.com/demo/accordion)**


= Features =
- **Fully Customizable**: All the options you need to arrange the showcase to your liking are available here.


= How to use =
- First, install the B Accordion plugin
- Add the Accordion block from the block category called "Widgets" in the Gutenberg editor.
- You can change block settings from the right-side settings sidebar.
- Enjoy!

* For installation help click on Installation Tab.

== Installation ==

= From Gutenberg Editor: =
1. Go to the WordPress Block/Gutenberg Editor
2. Search For **B Accordion**
3. Click on the **Accordion** to add the block

= Download & Upload: =
1. Download the **B Accordion** plugin (*.zip file*)
2. In your admin area, go to the Plugins menu and click on **Add New**
3. Click on **Upload Plugin** and choose the **`b-accordion.zip`** file and click on **Install Now**
4. Activate the plugin and Enjoy!

= Manually: =
1. Download and upload the **B Accordion** plugin to the **`/wp-content/plugins/`** directory
2. Activate the plugin through the Plugins menu in WordPress
